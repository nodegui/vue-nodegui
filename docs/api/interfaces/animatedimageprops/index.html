AnimatedImageProps

The animated image widget allows users to create and render native GIFs, video (without controls) elements in the app. It is based on NodeGui's QLabel along with NodeGui's QMovie

Usage#

<template>
<vn-view>
<vn-animated-image :src="'https://i.giphy.com/media/oYtVHSxngR3lC/giphy.gif'" />
</vn-view>
</template>
<script>
export default { }
</script>

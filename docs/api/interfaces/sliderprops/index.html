SliderProps

The slider widget allows users to render native sliders into their apps. It is based on the NodeGui's QSlider

Usage#

<template>
<vn-view>
<vn-slider v-model="sliderValue" :maximum="100" :minimum="10" :orientation="1" />
<vn-text>The slider value is: {{sliderValue}}</vn-text>
</vn-view>
</template>
<script>
import { ref } from 'vue';
export default {
setup() {
const sliderValue = ref(0);
return {
sliderValue
}
}
};
</script>
